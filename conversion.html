UTC → Metric Time Formula

Epoch: 2021-12-21 15:59:18 UTC — Unix 1640102358 — the closest December solstice to a whole UTC hour in the leap-second era (42 s from 16:00:00)
// UTC → Metric Time (pseudocode) // 1. How many real seconds have passed since the metric epoch? elapsed_real_seconds = current_unix_time - 1640102358 // 2. Convert to metric seconds (1 metric second = 0.864 real seconds) metric_seconds = elapsed_real_seconds / 0.864 // 3. Split into whole days and the time remaining in today day_number = floor(metric_seconds / 100000) // each metric day = 100 000 metric seconds time_today = metric_seconds mod 100000 // 0 – 99 999 // 4. Break the time-of-day into hours / minutes / seconds hour = floor(time_today / 10000) // 0 – 9 (10 hours per day) minute = floor(time_today mod 10000 / 100) // 0 – 99 (100 minutes per hour) second = time_today mod 100 // 0 – 99 (100 seconds per minute) // 5. Count off years from day_number // normal year = 365 days, leap year = 366 days // leap if: (year mod 4 = 0) AND (year mod 128 ≠ 0) year = (subtract year lengths from day_number until it goes negative) day_of_year = the remainder // just a number 0 – 364 (or 365 in a leap year) // ── Reverse: metric → UTC ────────────────────────────────────────────── unix_time = 1640102358 + (metric_seconds * 0.864)
